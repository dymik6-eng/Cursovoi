Данный проект релизован в целях курсового проекта.
И может быть запущен непосредственно из интепритатора либо при помощи файла .exe

Для запуска через интепритатор Python(версии 3.1 и выше) вам потребуется:
1. Установленная библиотека tkinter, модули: json, os, sys, tempfile
2. Создать проект
3. Пренести файлы: menu.py , game.py , clicker_save.json в папку с проектом
4. Для корректного запуска лучше запускать из файла menu.py

Для запуска пир помщи файла .exe:
1. Установите файл
2. Пернесите в удобную вам деррикторию
3. Запустите файл, если он в отдельной папке то папка с сохранением(clicker_save.json) создатся автоматически иначе файл сохранится в в временные файлы - temp
